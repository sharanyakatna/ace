ACE: Adaptive Confidence-Based Encoder Freezing

This repository contains the implementation of ACE (Adaptive Confidence-Based Encoder Freezing), a continual learning method designed to reduce catastrophic forgetting in task-incremental settings.

ACE adaptively freezes encoder layers based on model confidence, allowing the network to balance stability and plasticity across sequential tasks. The method operates without replay buffers, task identifiers, or model expansion.

This codebase supports the experimental work presented in the ACE research project and is intended for research and evaluation purposes.
